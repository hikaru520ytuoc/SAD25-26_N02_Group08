# Sprint 1 Summary

## Sprint name

Auth, User and Role Foundation

## Implemented scope

- JWT login API
- Current user API
- User schema, role schema, user-role mapping schema
- Audit log schema
- Password hashing with bcrypt
- JWT guard
- Roles guard
- `@Roles()` decorator
- `@CurrentUser()` decorator
- Admin user management API
- Role management API
- Audit log API for Admin
- Demo seed data for default roles and accounts
- Frontend login page
- Frontend dashboard page
- Frontend admin users page
- README and run guide update

## Out of scope

Sprint 1 does not implement topic registration, outlines, defense, reviewers, councils, scoring, revision or archive modules.

## Default roles

- ADMIN
- STUDENT
- SUPERVISOR
- REVIEWER
- COUNCIL_MEMBER
- COUNCIL_SECRETARY
- FACULTY_MANAGER
- ARCHIVE_STAFF
